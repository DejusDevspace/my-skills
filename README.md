# My Agent Skills

This repository is a personal collection of agent skills I have created for my
own workflows.

I keep these skills here so they are easy to revisit, improve, and share. The
repo is public so other people can look through the skills, adapt ideas that
resonate with them, or use the skills directly in their own agent workflows.

The skills reflect how I like agents to approach recurring tasks: with clear
processes, practical guardrails, and enough structure to make the work more
consistent without making it rigid.
